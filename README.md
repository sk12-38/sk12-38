## Hi there 👋

This GitHub profile is a place where I organize my finance and quantitative research projects using Python and data analysis.

Most of my projects are based on the Korean equity and ETF markets, covering topics such as factor strategies, index tracking, beta estimation, risk measurement, and empirical asset pricing paper replication. Each repository includes the analysis workflow, code, main results, and presentation materials where possible.

## Projects

| Project | Description |
| --- | --- |
| [Enhanced Index Tracking on KOSPI200](https://github.com/sk12-38/enhanced-index-tracking-kospi200) | A project on deep learning-based index tracking and enhanced index tracking strategies for the KOSPI200. |
| [Korean Equity Strategy Lab](https://github.com/sk12-38/korean-equity-strategy) | A collection of notebooks exploring and backtesting factor strategies in the Korean equity market. |
| [Bayesian Conditional Beta for KOSPI200](https://github.com/sk12-38/bayesian-conditional-beta-kospi200) | A Bayesian analysis of conditional beta and market regime changes for a KOSPI200 ETF. |
| [COVID-19 Tail Risk Transmission](https://github.com/sk12-38/covid-tail-risk-transmission) | An analysis of tail-risk transmission across ETF asset classes around the COVID-19 period using VaR, CVaR, GARCH, and CoVaR. |
| [Fama-French 1992 Korean Market Replication](https://github.com/sk12-38/ff1992-korean-market-replication) | A replication of Fama and French (1992) cross-sectional stock return analysis using Korean market data. |
| [MAX Anomaly Strategy](https://github.com/sk12-38/MAX_anomaly) | A strategy experiment on the residual MAX anomaly in the Korean equity market. |

## Topics

- Korean equity market analysis
- ETF and portfolio analysis
- Factor strategies and backtesting
- Beta estimation and market regime analysis
- Risk analysis using VaR, CVaR, GARCH, and CoVaR
- Empirical asset pricing paper replication
